# 💻 Programming Quiz

An interactive quiz platform developed to test and improve programming knowledge in a dynamic, educational, and accessible way.

---

## 📌 About the Project

The **Programming Quiz** is a web application that offers questions on different topics in the technology field, allowing users to assess their knowledge and learn in a practical way.

The project was developed focusing on:

* Active learning
* User engagement
* Simplicity and usability

---

## 🎯 Objective

To create an educational tool that:

* Encourages the study of programming
* Helps in the retention of content
* Offers immediate feedback to the user
* Makes learning more interactive

---

## 🧠 Features

* ❓ **Multiple choice questions**
* ⏱️ **Progress system during the quiz**
* ✅ **Real-time answer validation**
* 📊 **Final score at the end**
* 🔁 **Possibility to restart the quiz**
* 📚 **Questions focused on logic and programming**
